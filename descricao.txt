O tema escolhido foi um gerenciador de finanças. O sistema permite o controle de gastos mensais, registrando o mês, a descrição da despesa e o valor do gasto.

O sistema permite ao usuário cadastrar novas despesas, visualizar uma lista completa de gastos, consultar detalhes de uma despesa específica pelo seu ID, atualizar informações e remover gastos que não são mais necessários.

O CRUD foi aplicado pelas quatro operações essenciais: Create (Adiciona um novo dicionário, no caso os gastos à lista principal e salva no arquivo financas.json. O ID é gerado automaticamente baseado no tamanho da lista.)

Read (Lê o arquivo JSON ao iniciar e permite listar todos os dados ou um dado específico, lendo por ID.)

Update (Localiza um item pelo seu ID e altera os valores das chaves (mês, descrição ou valor) no dicionário, salvando a alteração no arquivo.)

Delete (Remove o item da lista baseado no ID fornecido e reescreve o arquivo JSON sem aquele registro.)
